# Copilot Chat — Workspace Prompts & Instructions

This file contains short prompts and preferences for Copilot Chat when working in this repository.

Preferred behavior
- Keep suggestions concise and focused.
- Prefer modern, accessible HTML and simple vanilla JavaScript.

Example prompts
- "Refactor `index.html` to be responsive and mobile-first, using semantic HTML only."
- "Add a contact form to `index.html` that validates client-side and shows a success message."
- "Improve accessibility in `index.html`: add ARIA attributes and fix heading order."

Custom quick prompts
- "Quick: create a responsive header + nav for this site."
- "Audit `index.html` for SEO and suggest meta tags."

How to use
1. Open Copilot Chat in VS Code and paste one of the example prompts.
2. Ask for step-by-step edits or a full patch.
3. When satisfied, apply changes and request tests or verification.
